1. The Problem
Project Overview
This project develops an interactive web application that transforms NYC CitiBike's complex operational data into actionable insights for better decision-making and system optimization.
a) What Problem Does Our Interactive App Solve?
NYC CitiBike system faces critical operational challenges in station balancing and resource management. Our interactive application addresses:
🚲 Station Imbalance
Some stations consistently empty while others overflow, causing user frustration and operational inefficiency
📊 Data Complexity
Millions of trip records with hidden temporal and spatial patterns difficult to analyze manually
🔍 Pattern Recognition
Complex usage patterns across time (hourly, daily, seasonal) remain invisible without proper tools
⚠️ Anomaly Detection
Unusual station behavior and system failures go undetected until user complaints
b) How Our App Solves These Problems
Transform complex bike-sharing data into actionable insights through interactive visualization, predictive analytics, and intelligent pattern recognition
and Make complex urban mobility data accessible, understandable, and actionable for everyone

3. Data Transformations
How We Processed the Raw Data
We took millions of raw bike trip records and transformed them into meaningful insights that can be easily visualized and understood.
📅 Daily Summaries
Convert individual trip records into daily arrival/departure counts per station
⚖️ Balance Calculations
Calculate net bike flow (arrivals minus departures) for each station daily
📍 Location Grouping
Group nearby stations by geographic clusters for neighborhood analysis
📊 Pattern Preparation
Aggregate data for time series analysis and trend visualization
🔄 Raw Trip Data → Daily Station Activity → Visual Patterns → Actionable Insights

4a. What Users Can Do & How It Helps
Interactive Features by Category
🗺️ Interactive Maps (Fully Interactive)
  • Click stations to see details and activity levels
  • Adjust radius to group nearby stations together
  • Filter by station type (busy, empty, balanced)
  • Benefit: Quickly identify problem areas needing attention
📈 Time Charts (Semi-Interactive)
  • Select date ranges to compare different periods
  • Hover for details on specific days or hours
  • Toggle between daily/hourly views
  • Benefit: Understand when demand peaks occur
🎯 Prediction Tools (Interactive Setup)
  • Choose any station from map or dropdown
  • Set forecast period (3-14 days ahead)
  • Adjust sensitivity for anomaly detection
  • Benefit: Plan ahead and catch problems early
🎨 Visual Analysis (Interactive Exploration)
  • Spider plots show multiple station features at once
  • Time wheels reveal daily and weekly patterns
  • Cluster analysis groups similar stations together
  • Benefit: Discover hidden patterns and relationships
💡 Practical Benefits
  • Decision makers get insights in minutes, not hours
  • Non-technical users can explore complex data
  • Findings can be shared easily with screenshots
  • Scales from small tests to city-wide analysis
